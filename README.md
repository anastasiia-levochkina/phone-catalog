# Phone Catalog

## 📌 Project Overview

This project is a **full-featured online phone catalog built with React and TypeScript**.
It allows users to browse different smartphone models, view detailed product information, and manage items in a shopping cart.

The application demonstrates modern frontend development practices including component-based architecture, routing, state management, and API integration.

## ▶️ How to Run Locally

Clone the repository:

```bash
git clone https://github.com/anastasiia-levochkina/phone-catalog.git
```

Navigate to the project folder:

```bash
cd phone-catalog
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm start
```

The project will run on a local development server.
